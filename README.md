
# Dental Website

A modern dental appointment booking website built with Next.js.

## Author

**Vatsal Kachhadiya**

## Features

- Dental-focused landing page with animations
- Expert cards with separate doctor detail pages
- Appointment booking form with modern UI
- Contact page and About Us page
- Dark/Light mode support
- Social footer with map embed

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- next-themes
- Radix UI + custom UI components

## Getting Started

Run the development server:

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
npm run start
```
